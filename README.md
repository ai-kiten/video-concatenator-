# 動画連結くん

ブラウザ上で動作する動画連結ツールです。キャッチ動画とボディ動画を自動的に組み合わせて連結します。

## 機能

- キャッチ動画（ファイル名に「キャッチ」または「冒頭」を含む）とボディ動画の自動組み合わせ
- BGM追加機能（MP3のみ対応）
- 再生速度調整（0.5倍〜2.0倍）
- 一括処理とZIPダウンロード
- 完全ブラウザベース（サーバー不要）

## 使い方

1. ブラウザでアプリケーションを開く
2. 動画ファイルをアップロード
3. 必要に応じてBGMや速度を設定
4. 「連結開始」をクリック
5. 完成した動画をダウンロード

## 技術スタック

- FFmpeg.wasm - ブラウザ上での動画処理
- JSZip - ZIP圧縮
- Vanilla JavaScript
